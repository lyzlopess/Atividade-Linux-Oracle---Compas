# Atividade-Linux-Oracle-Compas

## Pré requisitos

- Ter baixado na máquina o Oracle VM VirtualBox
- Ter baixado a imagem ISO

## Instalação do Linux Oracle
- Utilize a versão realease 8.6, Full ISO do Oracle Linux (Minimal).
- Escolha o Sistema Operacional sem Interface Gráfica.
- Configure no indioma inglês.

## Criação da VM

Com a VM instalada na máquina, abra na tela inicial e selecione a opção "Novo" para a criação da primeira VM, logo após, dê um nome e observe a pasta onde ficará os arquivos, selecionar o Tipo e escolher a opção "Linux" e selecionar a versão "Oracle-64bit".
